Модели - методы работы с БД

В папке entities находятся сущности завитсящие от класса Model.

Файл Repositories содержит CRUD-операции, т.е. общие методы для работы всех моделеймоделей.

Файлы, которые наследуются от Repositories и имеют в конце имени слово Repository, содержат частные методы, т.е. содержащие те же методы, что нужны для одноименных сущностей.
